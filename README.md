# 🎮 坦克大战 (Tank Battle)

经典FC游戏《坦克大战》(Battle City) 的 HTML5 重制版，纯前端实现，无需任何依赖。

## 🎯 游戏特色

- 🕹️ **经典玩法** - 还原FC坦克大战的经典体验
- 🧱 **可破坏地形** - 砖墙可被子弹摧毁，钢铁墙壁坚不可摧
- 🦅 **基地保护** - 保护底部的鹰标志，被摧毁则游戏失败
- 🎨 **多种敌人** - 普通坦克、快速坦克、重型坦克
- 💎 **道具系统** - 击杀敌人掉落道具（加速、护盾、冰冻、炸弹）
- 📈 **难度递增** - 关卡越高敌人越多越强
- 🗺️ **多种地图** - 3种不同布局循环出现

## 🎮 操作说明

| 按键 | 操作 |
|------|------|
| `↑` `↓` `←` `→` 或 `W` `A` `S` `D` | 移动坦克 |
| `空格键` | 发射子弹 |
| `P` | 暂停/继续 |

## 📝 游戏规则

1. 控制坦克消灭屏幕上所有敌方坦克
2. 保护基地（底部鹰标志），基地被毁则游戏失败
3. 每关需要消灭一定数量的敌人
4. 玩家有3条命，被敌人子弹击中损失一条命
5. 消灭所有敌人即可过关

## 🎨 关卡设计

游戏内置多个关卡，每关地图布局不同。随着关卡提升：
- 敌人数量增加
- 敌人移动速度变快
- 出现更多类型的敌人
